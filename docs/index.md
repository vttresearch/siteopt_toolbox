
# Siteopt tool

**Siteopt** is a tool for energy system optimization. The SiteOpt tool analyses future options of integrated energy systems in a local site with several energy vectors. On these pages you find the basic documentation of the tool. Use the navigator bar on the left to see an [Installation Guide](installation.md).  

You may also get familiar with **Siteopt web application** which provides a more user-friendly interface to casual users. Siteopt web application can be found on its [Github page](https://github.com/vttresearch/siteopt_app).
